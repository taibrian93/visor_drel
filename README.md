Proyecto DREL

1. Clonar repositorio:
    <br>
    git clone https://github.com/taibrian93/visordrel.git
2. Correr el siguiente comando:
    <br>
    composer install
3. Copiar .env.example:
    <br>
    cp .env.example .env
4. Correr el siguiente comando:
    <br>
    php artisan key:generate
5. Crear base de datos: 
    <br>
    sistemadrel
